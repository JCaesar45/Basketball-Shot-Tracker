# Basketball Shot Tracker

A simple web-based basketball shot tracker that allows users to log their made and missed shots, view their shooting percentage, and analyze their shooting performance over time. The application uses HTML, CSS, JavaScript, and Local Storage for data persistence.

---

## Features

- Log made and missed shots via interactive buttons
- Display total shots, made shots, missed shots, and shooting percentage
- View a chronological shot history with timestamps
- Data stored locally in the browser (Local Storage)
- Clean and responsive design inspired by NCAA March Madness theme

## How to Use

1. Clone or download the repository.
2. Open `index.html` in your web browser.
3. Use the **Made Shot** and **Missed Shot** buttons to log shots.
4. View updated stats and shot history below.

## Setup Instructions

### Prerequisites
- A modern web browser (Chrome, Firefox, Edge, etc.)

### Running Locally
1. Download or clone the project:
   ```bash
   git clone https://github.com/yourusername/basketball-shot-tracker.git
   ``
2. Navigate to the project directory:
   ```bash
   cd basketball-shot-tracker
   ``
3. Open `index.html` in your preferred web browser:
   - Double-click `index.html`
   - Or right-click and choose "Open with" your browser
